A design project for 'bauhaus|collectiv'; a theoretical brand that specialises in selling genuine bauhaus artefacts. A simplistic monochrome palette compliments a UI that wraps around the content, the side patterning takes inspiration from the typological form of the Joschmi font. Colour is used sparingly to highlight when the user mouseovers a product, this is done to give the user focus and to respect each individual artwork piece.

Branding

The bauhaus|collectiv logo combines two weights of the Recross Quadratic font to create a sleek and modern typographical brand.

The slogan features a fading Xants Regular font specifically chosen to emphasise the phrase “Own a piece of history”.

User Interface

This project was completed as an entry into the Adobe Hidden Treasures design competition.
